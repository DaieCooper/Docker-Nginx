1. Создаём Dockerfile в пустой директории
2. Создаём файл nginx.conf с конфигурацией Nginx в той же директории
3. Создаём директорию для хранения статических файлов при помощи команд:
mkdir static
echo "Hello, world!" > static/index.html
4. Создаём docker-compose.yml
5. Далее собираем образ из Докерфайла:
docker build -t my-nginx .
6. И запускаем докер - компоуз:
docker-compose up

Теперь на порту 80 будет работающий нжинкс, обслуживающий статический контент из директории static.Сайдкар lets encrypt будет создавать сертификаты в директории /etc/letsencrypt.
